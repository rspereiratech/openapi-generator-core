# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its Code of
Conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Summary

We pledge to make participation in this project a welcoming and respectful
experience for everyone, regardless of background or identity. Maintainers
and contributors are expected to act professionally and in good faith, both
in the repository (issues, pull requests, commit messages) and in any other
project space.

## Reporting

If you experience or witness unacceptable behaviour in a project space,
please report it privately to the maintainer at:

**rspereiratech@gmail.com**

Reports are handled confidentially. The maintainer will review the report,
follow up with the reporter, and take appropriate action in line with the
Contributor Covenant enforcement guidelines:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement>
